# ultimate-snake
1. Обучаем змейку и на 4 разных стадиях обучения сохраняем весы нейросети. Запустив ultimate_snake.py, вы сможете выбрать подходящую вам частоту смены кадров и сложность противника. Управление - WASD, игрок управляет зелёной змейкой.
2. Примечание: я убрал возможность сохранения весов из snake_train.py, потому что обучение идёт долго и веса лучше не перезаписывать. Тем не менее, процесс обучения можно увидеть, запустив snake_train.py и оставить симуляцию включённой на пару часов.
3. Требования: установленные numpy, tensorflow, keras и pygame
